# D20Masters 🎲

D20 Dice website for table top games like D&D, MTG, and more.

## Dungeon Roll Update Notes (March 2026)

This update adds a full gameplay expansion focused on variety, fairness, and replayability.

### Core Quality-of-Life

- Click-to-restart on end screen (defeat or victory) without using the restart button.
- Tile click bugs prevented during active gameplay by gating restart clicks to end-state overlay only.
- Safer monster spawn placement so enemies do not start right on top of the player.

### Combat and Enemy Improvements

- Elite monsters appear every 3rd level.
- Level 10 now contains a boss encounter with multiple HP phases.
- Combat now includes stronger feedback effects for hits, crits, and shield blocks.

### New Power and Progress Systems

- Chan's Shield can spawn on some levels and absorbs 2 monster hits.
- Combo streak tracking is now active.
- Score system added for kills, clears, and events.
- Loot drops can occur after kills:
  - Healing (+1 life, up to cap)
  - Battle Focus (+2 next combat roll)
  - Treasure bonus (+score)

### Level Variety

- Random room modifiers each level:
  - Classic
  - Haste Floor (faster monsters)
  - Blessed Floor (easier combat target)
  - Dark Floor (reduced visibility)
- Shrines can apply risk/reward effects:
  - Blessings (heal, shield refresh, combat bonus)
  - Curses (life loss)
- Event tiles can spawn:
  - Teleport Trap
  - Treasure Chest
  - Mimic ambush
  - Healing Fountain

### Win Route and Replayability

- Alternative route added: Perfect Clear.
- Clearing all monsters on a level auto-unlocks key progress and grants score bonus.

### Persistent Records

Run history now tracks and persists locally in browser storage:

- Best level reached
- Best streak
- Best score
- Total kills
- Total wins

### Balance Pass (March 2026)

- Reduced event overload and shrines to keep runs readable.
- Lowered random shield and loot frequency to preserve tension.
- Smoothed enemy scaling and speed curves across mid-level floors.
- Softened elite/boss roll pressure while keeping boss identity.
- Added streak-based bonus pacing (+1 next roll every 3-kill streak, capped).
