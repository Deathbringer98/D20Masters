# D20Masters Production Build

This is the final standalone React + Vite version of D20Masters.

No Tailwind dependency.
No PostCSS dependency.
No extra styling setup needed.

## Run locally

1. Extract the ZIP
2. Open a terminal in the project folder
3. Run:

npm install
npm run dev

4. Open the local URL shown in the terminal, usually:
http://localhost:5173

## Build for production

npm run build

## Preview production build

npm run preview
